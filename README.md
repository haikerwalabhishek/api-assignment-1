# API assingment 1

## Overview
This assignment evaluates the basic knowledge gained in Node.js and Express.js for building APIs, with a focus on performing CRUD operations.

## Features
- Add a user
- Get a user by ID
- Retrieve all users
- Update user information
- Delete a user
